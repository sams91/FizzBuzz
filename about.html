FizzBuzz

FizzBuzz is similar to another app called Hundo in that both apps emphasize the use of loops. In Hundo the numbers being displayed have two options, even numbers are displayed blue and bold while odd numbers are displayed red. This only required one If/Else statement to determin how those numbers would be displayed: “if number is divisible by 2 then even, else odd”.

The FizzBuzz app has 4 possible options to display so the If/Else statement is slightly more complex. First the app needs to determine if the number is a multiple of both Fizz and Buzz values, if so then “FizzBuzz” is displayed. If not then the app determines if the number is a multiple of the Fizz value, if so then “Fizz” is displayed. If not then the app determines if the number is a multiple of the Buzz value, is so then “Buzz” is displayed. If not then the number itself is displayed.

An interesting point of this app is found in the consequence of not determining if the value is a multiple of BOTH Fizz and Buzz first. If the app looked to see if the number was a multiple of just Fizz first and determined that it was then it wouldn’t bother looking to see if it was also a multiple of Buzz, it would just stop. The same holds true if multiples of Buzz were checked first. This would make it impossible for the app to distinguish between multiples of both and multiples of Fizz or Buzz individually, whichever was checked first.
