FizzBuzz

FizzBuzz is a Javascript coding exercise using loops. After you select a numeric value for Fizz and another for Buzz the program will display numbers from 1 to 100. Any number that is a multiple of the value entered for Fizz will be shown as “Fizz” in the display and any multiple of Buzz will be shown as “Buzz”. If the number happens to be a multiple of both values entered then it will display as “FizzBuzz”.

Instructions

Enter values for Fizz and Buzz. Then click the "Buzz It" button and the results will appear.
